/*【1号->2号】
位置信息：1、2、3、4、5、6、7、8

识别位置：1，5，3  (按识别顺序排列)

发送的信号：1，2，4，5，6，7，3（十六进制）  
解析：
①第1个位置：
有砝码：  发送1 -> 直接前进到4位置
没有砝码：发送2 -> 经2位置再前进到4位置

②第5个位置：
两个区域全有：   发送4 -> 直接前进到4位置
两个区域非全有： 发送5 -> 经6位置前进到位置

③第3个位置： 
两个区域全有：   发送6 -> 直接前进到8位置
两个区域非全有： 发送7 -> 经2位置前进到8位置

④抓完物品、放完物品后或者直接走，大车行进，发送3 -> 按规划的路径直接行进
*/
